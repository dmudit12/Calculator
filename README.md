# Calculator (C# Console Application)

A simple console-based calculator built using C#. Supports basic arithmetic operations with input validation.

---

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Input validation (prevents crashes on invalid input)
- Loop-based execution (continuous usage)

---

## Tech Stack

- Language: C#
- Runtime: .NET
- Interface: Console (CLI)

---

## How It Works

1. User inputs two numbers  
2. Selects an operation  
3. Program computes result  
4. Displays output  
5. Repeats until terminated
